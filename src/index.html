Tips:
1. To locate a brain region: type in its name in the "Search for a region" text box and it will be highlighted in the visualization

2. To search for connections between two regions: left click to set a region to be the source and shift + left click to set a region to be the target; then set the level for indirect connections using the "Max hop" slider. Click search to highlight the connections between the selected regions, or click clear to clear the highlighted connections

3. To adjust the level of details: use the "Max depth" slider
